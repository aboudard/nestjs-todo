## Description

NestJs Todo Application

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Swagger UI

http://localhost:3000/api/

## Request data

http://localhost:3000 - GET - Hello World!
http://localhost:3000/todo - REST api
